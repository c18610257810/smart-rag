# Smart RAG v1.0.0 Architecture Design

**版本**: 1.0.0  
**日期**: 2026-04-09  
**架构变更**: PGlite → Qdrant, 新增 RAG-Anything 集成

## 1. 架构概述

### 1.1 核心变更

| 组件 | v0.3.x | v1.0.0 |
|------|--------|--------|
| 向量存储 | PGlite (浏览器内) | Qdrant (外部进程) |
| 文档解析 | 仅 Markdown | RAG-Anything (全格式) |
| 数据容量 | <10GB | 150GB+ |
| 多模态 | ❌ | ✅ (图片理解) |

### 1.3 数据流

```
┌─────────────────────────────────────────────────────────────┐
│                        Indexing Flow                        │
└─────────────────────────────────────────────────────────────┘

Vault Markdown:
  Obsidian API → Text Extraction → Embedding → Qdrant (vault_notes)

Raw Folder (PDF/Word/PPT/Images):
  File Scan → RAG-Anything /parse → Chunks + Images + VLM Description
                                          ↓
                              Embedding → Qdrant (raw_documents, images)

┌─────────────────────────────────────────────────────────────┐
│                        Query Flow                           │
└─────────────────────────────────────────────────────────────┘

User Question:
  ↓
Embedding → Qdrant Search (vault_notes + raw_documents + images)
  ↓
Context Assembly (Top-K chunks + image descriptions)
  ↓
LLM Generate Answer
  ↓
Display with Citations
```

## 2. Qdrant Collections 设计

### 2.1 vault_notes (Vault 笔记)

存储 Obsidian Vault 内 Markdown 笔记的向量。

```json
{
  "id": "note_path_hash",
  "vector": [0.1, 0.2, ...],
  "payload": {
    "path": "Projects/Smart RAG/README.md",
    "title": "README",
    "content": "笔记内容...",
    "tags": ["project", "rag"],
    "modified_time": 1712649600,
    "word_count": 1500
  }
}
```

### 2.2 raw_documents (外部文档)

存储 raw folder 中 PDF/Word/PPT/Excel 等文档的 chunks。

```json
{
  "id": "doc_chunk_uuid",
  "vector": [0.1, 0.2, ...],
  "payload": {
    "file_path": "/Users/frank/Documents/Raw/2024_Report.pdf",
    "file_name": "2024_Report.pdf",
    "file_type": "pdf",
    "chunk_index": 3,
    "chunk_text": "文档内容...",
    "page_number": 5,
    "total_pages": 20,
    "file_hash": "sha256_hash",
    "indexed_at": 1712649600
  }
}
```

### 2.3 images (图片)

存储图片的 VLM 描述向量。

```json
{
  "id": "image_uuid",
  "vector": [0.1, 0.2, ...],
  "payload": {
    "description": "显示2024年Q1营收增长的柱状图，营收100万元，同比增长10%",
    "source_file": "/Users/frank/Documents/Raw/2024_Report.pdf",
    "page_number": 5,
    "extracted_path": "/tmp/extracted/img_001.png",
    "vlm_model": "gpt-4o-mini",
    "indexed_at": 1712649600
  }
}
```

### 2.4 chunks (通用 chunks)

通用文本 chunks，支持混合检索。

```json
{
  "id": "chunk_uuid",
  "vector": [0.1, 0.2, ...],
  "payload": {
    "text": "文本内容...",
    "source_type": "vault_note" | "raw_document",
    "source_id": "path_or_file_hash",
    "metadata": {}
  }
}
```

## 3. RAG-Anything HTTP Service

### 3.1 服务实现

文件: `rag_anything_server.py` (FastAPI)

```python
from fastapi import FastAPI, HTTPException
from pydantic import BaseModel
from typing import List, Optional
import asyncio
from raganything import RAGAnything, RAGAnythingConfig

app = FastAPI(title="RAG-Anything Service for Smart RAG")

# Global RAG instance
rag: Optional[RAGAnything] = None

class ParseRequest(BaseModel):
    file_path: str
    extract_images: bool = True
    extract_tables: bool = True

class ParseResponse(BaseModel):
    document_id: str
    chunks: List[dict]
    images: List[dict]
    tables: List[dict]
    metadata: dict

class QueryRequest(BaseModel):
    question: str
    top_k: int = 10
    mode: str = "hybrid"  # naive, local, global, hybrid

@app.on_event("startup")
async def startup():
    """Initialize RAG-Anything on startup"""
    global rag
    config = RAGAnythingConfig(
        working_dir="~/.openclaw/rag-storage",
        parser="mineru",
        enable_image_processing=True,
        enable_table_processing=True,
    )
    rag = RAGAnything(config=config)

@app.post("/parse", response_model=ParseResponse)
async def parse_document(req: ParseRequest):
    """Parse a document and return structured content"""
    if not rag:
        raise HTTPException(503, "RAG-Anything not initialized")
    
    result = await rag.aparse_file(req.file_path)
    
    return ParseResponse(
        document_id=result.document_id,
        chunks=result.chunks,
        images=result.images,
        tables=result.tables,
        metadata=result.metadata
    )

@app.post("/query")
async def query(req: QueryRequest):
    """Query the document library"""
    if not rag:
        raise HTTPException(503, "RAG-Anything not initialized")
    
    result = await rag.aquery(
        req.question,
        param=QueryParam(mode=req.mode, top_k=req.top_k)
    )
    
    return {
        "answer": result,
        "sources": result.sources if hasattr(result, 'sources') else []
    }

@app.get("/health")
async def health():
    """Health check"""
    return {
        "status": "healthy" if rag else "initializing",
        "rag_anything_version": "1.2.10"
    }
```

### 4.2 Core Classes

#### QdrantManager.ts

```typescript
export class QdrantManager {
  private process: ChildProcess | null = null;
  private dataDir: string;
  private httpPort: number = 6333;
  
  constructor(dataDir: string, port?: number) {
    this.dataDir = dataDir;
    if (port) this.httpPort = port;
  }
  
  async start(): Promise<void> {
    if (await this.isRunning()) return;
    
    const binaryPath = await this.getBinaryPath();
    this.process = spawn(binaryPath, [
      '--storage-path', this.dataDir,
      '--http-port', this.httpPort.toString()
    ]);
    
    await this.waitForHealthy(30000);
  }
  
  stop(): void {
    this.process?.kill('SIGTERM');
  }
  
  async isRunning(): Promise<boolean> {
    try {
      const response = await fetch(`http://127.0.0.1:${this.httpPort}/health`);
      return response.ok;
    } catch {
      return false;
    }
  }
  
  private async getBinaryPath(): Promise<string> {
    // Auto-download if not exists
    // Platform: darwin-arm64, darwin-x64, win32-x64
  }
}
```

## 7. Migration from v0.3.x

### 7.1 Data Migration

v0.3.x PGlite data → v1.0.0 Qdrant:

```typescript
async function migrateFromPGLite() {
  // 1. Export all embeddings from PGlite
  const notes = await pgLite.getAllNotes();
  
  // 2. Re-index to Qdrant
  for (const note of notes) {
    await qdrantClient.upsert('vault_notes', {
      id: hashPath(note.path),
      vector: note.embedding,
      payload: {
        path: note.path,
        content: note.content,
        ...
      }
    });
  }
}
```
